# Task Manager Application

A responsive task management web app with full CRUD operations and persistent storage using the Local Storage API. Built with modern ES6+ JavaScript, CSS Grid & Flexbox for a clean, mobile-first UI.

## Features
- Add, edit and delete tasks (full CRUD)  
- Persistent storage using Local Storage (data remains after refresh)  
- Real-time filtering and search of tasks  
- Task completion toggle and statistics (total, completed)  
- Modular ES6+ code: classes, arrow functions, template literals  
- Responsive design using Flexbox and CSS Grid

---

## Tech Stack
- HTML5  
- CSS3 (Flexbox, CSS Grid)  
- JavaScript (ES6+ classes, modules)  
- Local Storage API
